# Getting Started:
### Add the vagrant box if you do not already have it: 
* vagrant box add pussinboots/ubuntu-truly-full

### Run this: 
git clone https://github.com/joshmccall221/dotvagrant.git && cd ./dotvagrant && vagrant up
